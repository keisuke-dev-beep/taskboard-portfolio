# My TaskBoard

Next.js / TypeScript / Tailwind CSS を使って作成したシンプルなタスク管理アプリです。

短期間でポートフォリオとして形にすることを目的に、
まずはタスクの追加・一覧表示・完了切替・削除といった基本機能を中心に実装しました。

## できること
- タスクの追加
- タスク一覧表示
- タスクの完了切替
- タスクの削除

## 使用技術
- Next.js
- TypeScript
- React
- Tailwind CSS
- API Routes

## 工夫した点
- コンポーネントを分けて、画面ごとの役割が分かりやすくなるようにした
- API Routesを使い、フロントとAPIの処理を分けた構成にした
- 余白や文字サイズを調整して、シンプルで見やすいUIになるようにした
- 短期間での作成だったため、データはメモリで管理する形にしている

## 補足
実務ではJava（Spring Boot）とAngularを中心に開発していましたが、
